# Jacob Bailly — Portfolio

Personal portfolio site built with React.

🔗 **Live:** [jacobbailly123.github.io/portfolio](https://jacobbailly123.github.io/portfolio)

## Sections

- **Hero** — Introduction and call to action
- **About** — Background, education, and research
- **Projects** — SociaLink, Wordle Clone, SKEF Research
- **Skills** — Technical skills and certifications
- **Contact** — Contact form and direct links

## Tech Stack

- React 18
- CSS-in-JS (inline styles)
- Google Fonts (DM Serif Display + DM Sans)
- Deployed via GitHub Pages

## Local Development

```bash
npm install
npm start
```

## Deployment

```bash
npm run deploy
```
